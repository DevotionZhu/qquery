    A basic query engine for pyqgis

    Project maintained by NathanW2 Hosted on GitHub Pages — Theme by mattgraham

    A mini query engine for pyqgis to query features from a layer.

    It defines a simple SQL style API to create a query

    q = (query(layer).where("postcode > 6164 OR postcode < 6167")
                 .select('assessment', 'address', 'lot')()

    Results are returned via a generator so results are returned as needed.

    q.next()
q.next()

    Normal Python functions can be used in a .select() and .where()

    def checkassessment(feature):
    if feature['assessment'] is None:
        return False

    return int(feature['assessment']) == 4315968
